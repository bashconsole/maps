1 Назначение программы

Программа предназначена для создания файла конфигурации драйвера сети МАПС. Программа выполнена в виде мастера.

2 Конфигурирование сети МАПС

Конфигуратор состоит из шести страниц ввода параметров сети. Переход от одной странице к другой осуществляется кнопками [Next > ] - для перехода к следующей странице конфигуратора и [ < Back] - для перхода к преведущей. Переход на следующую страницу возможен в том случае, если были определены те параметры конфигурирования, которые не заданы по умолчанию. Переход на преведущую страницу возможен всегда, за исключением первой страницы конфигуратора. Генерация файла конфигурирования сети МАПС осуществляется нажатием кнопки [Finish], которая находится на последней странице конфигуратора. Вызов помощи осуществляется кнопкой [Help].
В качестве параметра конфигуратору может быть передано имя файла ранее созданной конфигурации данные которого и будут начальными значениями при редактировании. Например
[root@host /usr/local/bin]# mapsconfig mapsconf
где mapsconf - ранее созданный файл конфигурации.

2.1 Выбор пути к файлам для загрузки КСв-31

В данной секции конфигурирования необходимо указать путь к файлам для загрузки КСв-31, то есть директорию, в которой находятся файлы с расширением *.hex. Указать директорию можно двумя способами: выбрать директорию в окне выбора директории мышкой или клавишами курсора или ввести путь к файлам в строке ввода, при этом дерево каталогов будет разворачиваться по ходу выбора директории, что обеспечивает наглядность процесса выбора. При вводе правильного пути к файлам (к каталогу с файлами с расширением *.hex) кнопка [Next > ] станет доступной.

2.2 Конфигурирование магистралей

В этой секции конфигурирования определяется количество магистралей, адрес порта подключения устройства КСв-31 для каждой магистрали и номер уровня прерывания для подключения устройства КСв-31. Количество магистралей не может быть меньше одной и больше трех. Чтобы удалить или добавить магистраль в конфигурацию сети МАПС воспользуйтесь опцией "Доступна". При отключении магистрали, такие параметры настройки данной магистрали как адрес порта подключения и номер уровня прерывания становятся не доступными для редактирования.

2.3 Абоненты и режим функционирования КСв-31

Данная секция состоит из двух частей. Секция конфигурирования абонентов сети МАПС включает определениетаких параметров функционирования как
  • маскимальный адрес подключения абонента в сеть, который не может превышать 62 (по умолчанию 62)
  • адрес подключения данной ПЭВМ в сеть, который не может превышать аскимальный адрес подключения абонента в сеть
Секция конфигурирования режима функционирования КСв-31 как арбитра в сети МАПС. Для каждой определенной ранее магистрали может быть задан только один из вариантов:
  • не арбитр
  • единственный арбитр
  • резервный арбитр магистрали
  • резервный арбитр сети
При конфигурировании режима функционирования магистралей не может быть только одна из нескольких доступных магистралей определена как "резервный арбитр сети". В этом случае все определенные магистрали должны быть резервными арбитрами сети. Если магистраль не доступна (отключена), то определение магистрали как арбитра также будет не доступно.

2.4 Конфигурирование параметров функционирования КСв-31

Секция конфигурирования параметров функционирования КСв-3 состоит из двух частей. В секции "Параметры" осуществляется конфигурирование параметров функционирования КСв-31, таких как
  • приоритетность ПЭВМ как абонента: приоритетный - "да", неприоритетный - "нет" (по умолчанию "нет")
В секции "Квоты параметров функционирования" определяются такие параметры функционирования КСв-31
  • время цикла опроса сообщений (по умолчанию 10)
  • время ожидания ответа (по умолчанию 10)
  • квота выдачи сообщения для приоритетных абонентов (по умолчанию 5)
  • квота выдачи сообщения для приоритетных абонентов при исчерпаниии длительности цикла (по умолчанию 5)
  • квота выдачи сообщения для неприоритетных абонентов (по умолчанию 5)

2.5 Параметры приема/выдачи сообщения

На данном этапе конфигурирования сети МАПС определяются такие параметры функционирования как
  • квота приема сообщений без подтверждения (по умолчанию 37)
  • квота приема сообщений с подтверждением (по умолчанию 5)
  • квота выдачи сообщений без подтверждения (по умолчанию 37)
  • квота выдачи сообщений с подтверждением (по умолчанию 5)
  • максимальное количество одновременно открытых точек доступа к сети (по умолчанию 1)
  • максимальное количество буферов для приема сообщений из сети для каждой точки доступа (по умолчанию 1)
  • имя файла конфигурации (по умолчанию mapsconf)

2.6 Генерация файла конфигурирования сети МАПС

Данная секция конфигурирования является завершающей. В этой секции предоставляется просмотр всех параметров, значения которых были определены в процессе работы с конфигуратором. Если необходимо изменить один из параметров, следует вернуться на страницу определения значения данного параметра и отредактировать его. Если введенные данные удовлетворяют требованиям пользователя, следует нажать кнопку [Finish], в результате чего, будет создан файл конфигурирования сети МАПС, который будет находиться в каталоге, который был задан как каталог с файлами для загрузки КСв-31.
